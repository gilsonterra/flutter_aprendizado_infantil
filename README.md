### Atualizar dependencias

```
flutter pub get
```

## Executar aplicação

```
flutter run
```

## Executar aplicação em modo debug

```
flutter run --debug
```

## Executar aplicação em modo release

```
flutter run --release
```

## Limpar cache

```
flutter clean
```


### Emular dispositivo

```
flutter emulators --launch Medium_Phone_API_36.0
```
